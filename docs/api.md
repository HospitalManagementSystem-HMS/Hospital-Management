# HMS API (via API Gateway)

Base URL (frontend uses same origin): `/api`

Auth:

- Use `Authorization: Bearer <JWT>` for protected endpoints.
- Roles: `ADMIN`, `DOCTOR`, `PATIENT`

## Auth

### POST `/api/auth/register` (Patient)

Body:
```json
{ "email": "patient@example.com", "password": "Password@123", "name": "Patient Name" }
```

Response:
```json
{ "token": "<jwt>", "user": { "id": "<authUserId>", "email": "patient@example.com", "role": "PATIENT" } }
```

### POST `/api/auth/login` (All roles)

Body:
```json
{ "email": "admin@hospital.local", "password": "Admin@12345" }
```

### GET `/api/auth/me` (Protected)

Response:
```json
{ "user": { "id": "<authUserId>", "email": "…", "role": "…" } }
```

## Users / Doctors

### GET `/api/doctors` (Public)

Response:
```json
{ "doctors": [{ "id": "<doctorAuthUserId>", "name": "Dr…", "specialization": "…", "email": "…" }] }
```

### GET `/api/users/me` (Protected)

Returns the role-specific profile (doctor/patient) when available.

### POST `/api/admin/doctors` (Admin)

Body:
```json
{ "name": "Dr. A", "specialization": "General Medicine", "email": "dr.a@hospital.local", "password": "TempPass@123" }
```

Response:
```json
{ "doctor": { "id": "<authUserId>", "name": "…", "email": "…", "specialization": "…" } }
```

## Appointments

Status enum: `PENDING | ACCEPTED | REJECTED | COMPLETED`

### POST `/api/appointments` (Patient)

Body:
```json
{ "doctorId": "<doctorAuthUserId>", "startTime": "2026-04-24T10:00:00.000Z", "durationMinutes": 30 }
```

Errors:
- `409 SLOT_UNAVAILABLE` when overlapping slot exists for the doctor.

### GET `/api/appointments/me` (Patient)

Lists patient appointments (includes `doctorEmail` when available).

### GET `/api/doctor/appointments` (Doctor)

Lists doctor appointments (includes `patientEmail` when available).

### PATCH `/api/appointments/:id/decision` (Doctor)

Body:
```json
{ "status": "ACCEPTED" }
```

Allowed only when appointment is `PENDING`.

### POST `/api/appointments/:id/consultation` (Doctor)

Completes an accepted appointment and stores prescription + follow-up.

Body:
```json
{
  "medicines": [{ "name": "Paracetamol 500mg", "instructions": "1-0-1 after food" }],
  "notes": "Hydration + rest",
  "followUpRecommended": true,
  "followUpDate": "2026-04-30T10:00:00.000Z"
}
```

### GET `/api/admin/appointments` (Admin)

Lists recent appointments for monitoring.

## Notifications

### GET `/api/notifications` (Protected)

Response:
```json
{ "notifications": [{ "_id": "...", "type": "APPOINTMENT_STATUS", "message": "...", "readStatus": false, "createdAt": "..." }] }
```

### POST `/api/notifications/:id/read` (Protected)

Body:
```json
{ "read": true }
```

## Admin Activities

### GET `/api/admin/activities` (Admin)

Shows recent activity events generated by services.
